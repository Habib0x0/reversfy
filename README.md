# Installation 
``` 
git clone https://github.com/hab1b0x/reversfy.git 
cd reversfy && bundle install 
```

# Usage
``` 
ruby reverse.rb 

 Usage:                                  
   - Enter the IP address of the listener  
   - Enter the port number of the listener 
   - Select the shell
```
